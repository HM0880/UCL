# GitHub markdown basics [x](https://help.github.com/articles/markdown-basics/)

### Markdown summary
- Headings: `# TEXT`, `## TEXT`, ... `###### TEXT`
- Blockquotes: `> TEXT`
- Bold: `**TEXT**`
- Italic: `*TEXT*`
- Unordered list: `*  TEXT' or `-  TEXT`
- Ordered list: `1. TEXT`
- Inline formatting: ` `TEXT` `
- Links: `[TEXT TO DISPLAY](LINK)`
